# AI context packs

A context pack is the clean block you give to Cursor, Codex, Claude, or another assistant.

## Try this now

```bash
memory dev context --for codex
memory dev context --for cursor
memory dev context --for claude
```

## What it includes

- Project summary.
- Current branch and dirty file count.
- Important files.
- Commands to run.
- Recent decisions and fixes.
- Known pitfalls from memory.

## Why this helps

The assistant starts with repo-specific facts instead of guessing from a cold prompt.
