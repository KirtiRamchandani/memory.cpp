# FAQ

## Is this ChatGPT memory?

No. ChatGPT memory is personal. memory.cpp is repo memory.

## Does it upload my code?

No cloud upload is required by the core tool.

## Does it replace Git?

No. Git stores diffs. memory.cpp stores meaning, context, and next steps.

## Does it require Ollama or OpenAI?

No. The default provider is local hash search. Fastembed-style local semantic hashing is also built in.

## How do I turn it off?

Stop runtime processes:

```bash
memory stop
```

Delete data:

```bash
memory privacy purge --yes
```

## Is terminal memory automatic?

No. It is opt-in.
