# Onboard a new developer

Goal: create a private-safe handoff bundle.

```bash
memory handoff new-dev --output .memory.cpp/handoff
memory share onboarding --output .memory.cpp/share/onboarding-brief.md
memory adoption checklist
```

Expected output: Markdown handoff files and a local adoption checklist.

What happened: memory.cpp gathered local project summary, commands, decisions, pitfalls, and recent changes.

Privacy note: review handoff files before sharing.
