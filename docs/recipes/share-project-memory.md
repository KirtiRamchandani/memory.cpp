# Share project memory

Goal: create a safe artifact for a README, issue, or chat.

```bash
memory share status --output .memory.cpp/share/project-memory-summary.md
memory share map --output .memory.cpp/share/project-evolution-map.html
memory share context --private-safe
```

Expected output: local Markdown/HTML files with redacted content.

What happened: memory.cpp summarized local repo memory and wrote shareable files.

Privacy note: review artifacts before posting them anywhere.
