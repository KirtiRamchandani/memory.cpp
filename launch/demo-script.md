# Demo script

1. Install dry run: `./scripts/install.sh --dry-run`
2. Setup: `memory setup --developer --yes`
3. Morning brief: `memory dev morning`
4. AI context: `memory context write --for cursor`
5. Map: `memory map --type evolution --output html`
6. PR: `memory pr summary --base main`
7. Handoff: `memory handoff new-dev --output .memory.cpp/handoff`
