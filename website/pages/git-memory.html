Git Memory

Turn local commits, branches, and diffs into explainable memory.

memory setup --developer
memory dev morning
memory dev context --for codex
memory show-map

Try this now

memory what
memory where
memory status
memory doctor

These commands explain what is stored, where it lives, whether setup is healthy, and what to do next.

Expected output

saved locally: .memory.cpp/memory.db
not uploaded anywhere
next: memory dev next

The CLI starts with a short summary and gives exact follow-up commands.

How this fits the product promise

memory.cpp helps a repo remember what happened, why it changed, and what to do next. This page focuses on practical developer and AI-coding workflows only.
