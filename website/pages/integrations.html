AI coding tools

Attach safely.

Use memory.cpp with Cursor, Claude, VS Code, Codex, Continue, and Ollama while keeping MCP read-only by default.

$ memory attach cursor --dry-run
$ memory attach --print-config cursor
$ memory attach cursor --yes
$ memory attach status

Cursor

Writes `.cursor/mcp.json` with a local MCP server.

memory attach cursor --dry-run

Claude

Generates a Claude Desktop-style config snippet for local use.

memory attach claude --dry-run

VS Code

Creates `.vscode/mcp.json` for compatible editor tooling.

memory attach vscode --dry-run

Codex

Prefer context packs when direct attach is not available.

memory context write --for codex

Continue

Creates a local `.continue/mcp.json` snippet.

memory attach continue --dry-run

Ollama

Prints proxy setup without starting long-running services by surprise.

memory attach ollama --dry-run
safety

Safe defaults

Read-only MCP tools are enabled by default. `memory_add`, `memory_patch`, and `memory_forget` require explicit approval or remain disabled in generated snippets.

memory detach cursor --dry-run
memory privacy status
memory attach doctor
